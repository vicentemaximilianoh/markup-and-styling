Block vs Inline vs Inline-block

Block

  • Width and height are allowed to set
  • all margin sides are allowed to set
  • adds a line break after an element

Inline

Block 1 Block 2 Block 3 Block 4
  • Width and height are NOT allowed to set
  • top and bottom margin are NOT allowed to set
  • does NOT add a line break after an element

  • Width and height are allowed to set
  • all margin sides are allowed to set
  • does NOT add a line break after an element

Transitions

A property and a duration of the transition must be specified at least. Also timing linear function and delay time can be specified. To transition more that one property, they must be separated by comma.

Columns Layout

Reference

Column-width is the minimum width of the column. If column-count is set then browser automatically adjust number of columns provided unless width of columns is less that column-width property value.

Colors

  • Hexa: #000000 - #ffffff
  • RGB: red - green - blue => rgb(0-255, 0-255, 0-255)
  • RGBA: red - green - blue - alpha channel => rgba(0-255, 0-255, 0-255, 0.0-1)
  • HSL: hue - saturation - lightness => hsl(0-360 deg, 0-100%, 0-100%)
  • HSLA: hue - saturation - lightness - alpha channel => hsla(0-360 deg, 0-100%, 0-100%, 0.0-1)
  • color names: black, white, green, and so on.
