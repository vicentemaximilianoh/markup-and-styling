Box model and box-sizing property.

Content box = content + padding = 520 x 120 = padding and border is NOT included in width and height sizes
Content border = 500 x 100 = padding and border is included in width and height sizes

Transitions

A property and a duration of the transition must be specified at least. Also timing linear function and delay time can be specified. To transition more that one property, they must be separated by comma.
